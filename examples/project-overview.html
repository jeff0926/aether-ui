🚀 What We Built

Aether UI inverts the traditional UI paradigm. Instead of shipping JavaScript bundles to hydrate virtual DOM trees, we project pre-computed HTML slivers via Server-Sent Events to a minimal kernel. The browser becomes a projection surface, not an application runtime.

Multi-agent coordination hub for sharing state across Aether runtime instances. Singleton pattern with pub/sub.

Size: 837 bytes raw • 442 bytes gzipped
File: src/kernel/aether-orchestrator.js

Web Component Complete

Custom <aether-runtime> element for declarative usage. Handles endpoint, namespace, mood attributes.

Integration layer for existing frameworks. Supports Snap-In (coexist), Jump-In (replace), and Net-New (pure) modes.
