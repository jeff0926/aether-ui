# Aether UI

Zero-hydration interface framework for agentic systems.

## Overview

Aether UI inverts traditional UI frameworks: the browser is a projection surface, not an application runtime. The agent owns the interface DNA—structure, style, and logic live in the Knowledge Graph. The browser receives pre-computed HTML slivers via SSE and renders them through a 2KB kernel.

## Quick Start

```bash
npm install
npm run build
npm run dev
```

## Features

- **2KB kernel** - Zero-hydration, SSE-based projection
- **3-mode adapter** - Snap-in, Jump-in, Net-new integration
- **Edge-native** - Cloudflare Workers, Vercel Edge ready
- **Design system agnostic** - CSS variable possession
